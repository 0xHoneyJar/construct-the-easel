# The Easel

> Creative studio for visual direction — moodboarding, reference collection, and design decisions.

## Skills

| Command | What it does |
|---------|-------------|
| `/grounding-creative` | Set up the visual vocabulary for a design area — review what exists, identify gaps, prepare for exploration |
| `/exploring-visuals` | Generate focused image prompts grounded in your project's aesthetic. Works with AI image tools (Recraft, Fal.ai) |
| `/capturing-results` | Annotate visual results — what worked, what didn't, which aesthetic directions to keep |
| `/recording-taste` | Record a design decision with context, rationale, and alternatives considered |

## When to use

- Starting a new visual direction and need to collect references and define the aesthetic
- Generating AI images and want prompts grounded in your project's existing visual language
- Making a design decision and want to document why, so the team stays aligned
- Reviewing visual output and need to systematically capture what works
